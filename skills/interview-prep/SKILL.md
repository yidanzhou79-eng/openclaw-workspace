# Interview-Prep 技能

专业的面试准备系统，提供完整的面试流程模拟和个性化辅导。

## 功能特性

### 1. 面经搜索
- 自动搜索目标公司和岗位的面经
- 智能提取面试情报（常见问题、流程、难点等）
- 多维度搜索关键词组合

### 2. 模拟面试
- 基于公司风格和岗位要求生成问题
- 智能难度调整（根据用户表现）
- 避免重复问题（除非是弱项领域）

### 3. 智能评估
- 多维度评分（技术准确性、深度、结构、批判性思维）
- 客观错误检测
- 个性化改进建议

### 4. 记忆管理
- 用户档案管理
- 弱项追踪和强化
- 进步历史记录
- 个性化适配

## 使用方式

```
# 开始新面试准备
interview-prep new [公司] [岗位]

# 开始模拟面试
interview-prep simulate [难度]

# 提交回答并获取评估
interview-prep evaluate [回答内容]
```

## 数据存储

所有数据存储在 `INTERVIEW_DATA/` 目录下：
- `users/` - 用户档案
- `interview_sessions/` - 面试会话记录
- `knowledge_base/` - 知识库
